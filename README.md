# MyWMS
中文 | English

一个基于现代技术栈构建的、功能全面、易于部署的仓库管理系统（WMS）。它旨在帮助中小型企业高效、精准地管理仓库的日常作业，实现库存可视化与流程标准化。

✨ 核心特性
📦 库存管理：多维度库存查询、实时库存跟踪、库存预警与盘点。

📋 入库管理：支持采购入库、退货入库等，流程化指引，支持盲收与明收。

🚚 出库管理：支持销售出库、调拨出库等，多种拣货策略（如FIFO、FEFO），波次计划。

🔀 库内管理：库位管理、库存调拨、库存冻结/解冻、包装管理。

📊 报表与分析：丰富的库存报表、出入库流水、操作日志，助力数据驱动决策。

👨‍💼 权限控制：基于角色的权限访问控制（RBAC），精确到按钮级别的操作权限。

📱 条码/RFID支持：全面支持条码扫描枪和RFID设备，提升作业效率与准确性。
